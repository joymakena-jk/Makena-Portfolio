ABOUT

A bit about my approach.

I'm a front-end developer who builds responsive websites from the ground up using HTML, CSS, and JavaScript — no page builders, no templates. I like matching a site's visual language to the brand it's built for, so a luxury hotel doesn't feel like a gaming platform, and a coffeehouse doesn't feel like an airline.

Every project on this page includes real interactivity — booking flows, live search, dynamic content, form handling — not just a static layout. I care as much about the code being clean and accessible as I do about the design looking intentional.
